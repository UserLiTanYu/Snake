# Neon Snake (霓虹贪吃蛇)

一个基于 C++ NDK 和 OpenGL ES 3.0 打造的 Android 贪吃蛇游戏。本项目重点展示了如何通过原生代码实现丝滑的动画效果、高性能的图形渲染以及复杂的 JNI 交互。

## 🌟 项目特性

- **高性能渲染**：采用 OpenGL ES 3.0 编写着色器，支持发光 (Glow) 效果、圆角矩形渲染以及平滑的抗锯齿处理。
- **流畅的操控体验**：
    - **双摇杆布局**：左侧虚拟摇杆控制方向，右侧按钮控制加速。
    - **路径平滑算法**：通过 `pathHistory` 记录蛇头轨迹，实现身体跟随的丝滑曲线效果，而非传统的网格移动。
- **多状态管理**：包含 开始界面、模式选择（无尽、挑战、更多）、游戏进行中、暂停以及游戏结束状态。
- **原生资源交互**：
    - 动态生成文字贴图（通过 Java 层 Canvas 绘制并传递给 C++）。
    - NDK 直接加载 Assets 贴图资源。
    - 深度集成声音系统（BGM 与 SFX 分离管理）。
- **响应式 UI**：纯 C++ 编写的 UI 组件库，支持自动适配屏幕比例的按钮和背景图。

## 🛠️ 技术栈

- **语言**：C++ 17, Kotlin/Java
- **图形 API**：OpenGL ES 3.0
- **构建工具**：CMake, Gradle
- **库依赖**：
    - `androidx.games:games-activity` (Game SDK)
    - `GLM` (用于矩阵计算，位于 `Utility.h`)
    - Android NDK (Native Development Kit)


## 🚀 快速开始

### 环境要求
- Android Studio Ladybug (或更新版本)
- Android SDK Platform 34+
- NDK 27.0.12077973+
- CMake 3.22.1+

### 编译运行
1. 克隆本项目。
2. 在 Android Studio 中打开项目。
3. 等待 Gradle 同步完成。
4. 点击 **Run** 按钮部署到 Android 真机或模拟器（建议使用支持 OpenGL ES 3.0 的真机）。

## 🎮 玩法说明

1. **开始游戏**：点击主界面的“开始游戏”进入模式选择。
2. **选择模式**：目前支持“无尽模式”，其他模式正在开发中。
3. **操作**：
    - **左手**：拖动左下角摇杆控制蛇的转向。
    - **右手**：长按右下角橙色按钮进入加速模式（会消耗少量分数或长度）。
4. **目标**：收集发光的彩色食物增加长度和得分，避免碰撞到自己的身体或边界（取决于模式设置）。
