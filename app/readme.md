可扩展的功能
1.添加背景音乐（至少一种），音效（至少三种，点击ui音效，蛇吃食物音效，蛇加速音效）
2.优化设置按钮，设置按钮先设置四个设置项，音乐开关，音效开关，退出游戏，返回（游戏中也可以点击设置按钮，将退出游戏替换为返回主界面）
3.优化当前无尽模式玩法，可以添加一些ai蛇，修改一下背景图，优化一下蛇的外观
4.添加一些新的食物和道具，比如加速器 、护盾、磁铁。获取道具后得到短暂的增益效果。
5.在无尽模式玩法左上角添加一个可折叠的框，显示当前游戏的实时排名，按照蛇的长度排名
6.完成挑战模式的开发，可以设置多个关卡，每个关卡设置一个目标，完成目标则挑战成功，可以挑战下一关。
7.完成更多玩法模块的开发，可以自己思考做什么模式
